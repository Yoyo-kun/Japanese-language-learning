## 日语构成

>   ![image-20241017224118897](https://s2.loli.net/2024/10/17/xL3jBTINMp1KuW9.png)

-   **当用汉字**：昨日、買
-   **罗马字/音**：SONY
-   **假名**： 【其余部分】

****

### 汉字

-   **形同意同**：日本	学校	教室	山	水
-   **形同意异**：人参【胡萝卜】	手紙【信，信件】	娘【女儿】
-   **形异意同**：散歩	圧力	具体
-   **形异意异【自造字】**：辻【十字路口】	畑【旱田】	峠【山顶】

****

### 罗马字

-   **日文录入**
-   **辅助发音**：TOYOTA
-   **书写商标**
-   **名片**：日常中，交换名片时，会写上罗马音以避免读错名字

****

### 假名

-   **平假名**：由 **草书** 演化而来
-   **片假名**：由 **楷书偏旁部首** 演化而来

#### 平假名的用途

-   **日语文字构成之一**
-   **给汉字注音**【罗马字 --注音--> 平假名 --注音--> 当用汉字】
-   **文法助词的使用**【日语是黏着语，通过助词和后缀来表示语法关系，而不是依赖词序】

#### 片假名的用途

-   **书写外来语**
-   **拟声拟态词**
-   **电报用语**
-   **表示强调、郑重**
-   **动植物、矿物质**

-   日本语中使用**行、段**，而非汉语的**行、列**。
-   日本语中只有**单元音**【如：a、e、i、o、u 等，这也是在说，发音时，舌位、口型应当保持不变】，没有**双元音**【如：ei、ai 等】，更没有**大舌音**、**小舌音**等。
-   第一行为**元音行**，其余行为**辅音行**。
-   辅音行与元音行进行组合发音。
-   最后独立于前面各行的单独音为**拨音**，不单独发音。

### 「あ」行假名

#### あ (a) / ア (a)

| <img src="https://s2.loli.net/2024/10/18/rDhSHtG2UWJF1wz.png" width="200px" height="200px"> | <img src="https://s2.loli.net/2024/10/18/ExcR3YCPAjKSZ6s.png" width="200px" height="200px"> |
| ------------------------------------------------------------ | ------------------------------------------------------------ |

**字源**：「あ」【安】	「ア」【阿】

**发音**：「あ」日本语中开口最大，舌位最低，双唇自然张开，比发“啊”的口型略小，震动声带，声音洪亮。【扁唇音】

**书写**：「あ」的底部是水平的；「ア」的下一笔几乎是「竖撇」。

****

#### い (i) / イ (i)

| <img src="https://s2.loli.net/2024/10/18/LUpqdKywgcStjIb.png" width="200px" height="200px"> | <img src="https://s2.loli.net/2024/10/18/ioAyWaY1rESFCRV.png" width="200px" height="200px"> |
| ------------------------------------------------------------ | ------------------------------------------------------------ |

**字源**：「い」【以】	「イ」【伊】

**发音**：「い」的发音比发“衣”时，双唇略松，嘴角裂开较小。前舌隆起，和硬腭相对，形成狭窄通道，前舌用力，振动声带，声音较尖。【扁唇音】

**书写**：「い」的中间想象有一个椭圆，用左右两个笔画包围，顶部水平；「イ」正常书写。

**示例**：

-   愛（<span style="text-decoration: overline;">あ</span>い）**【有上横线标记的为高音】**
-   胃（い）

****

#### う (u) / ウ (u)

| <img src="https://s2.loli.net/2024/10/18/ITb6gpB87XwWJY2.png" width="200px" height="200px"> | <img src="https://s2.loli.net/2024/10/18/AHY7SWgsbNuiwjJ.png" width="200px" height="200px"> |
| ------------------------------------------------------------ | ------------------------------------------------------------ |

**字源**：「う」【宇】	「ウ」【宇】

**发音**：「う」的发音双唇自然微启，嘴角微拉，双唇不要发像“乌”那样向前突出【如果没有感觉，可先发「あ」、「い」的音，并保持嘴唇不动，然后发声。】振动声带，声音较弱。【扁唇音】

**书写**：「う」第二笔较为圆滑，需要有一定弧度；「ウ」第一、二笔是短竖，而非点。

****

#### え (e) / エ (e)

| <img src="https://s2.loli.net/2024/10/18/Ou5ZIlwitsjVgAr.png" width="200px" height="200px"> | <img src="https://s2.loli.net/2024/10/18/4JqENtk2HjyP5Ud.png" width="200px" height="200px"> |
| ------------------------------------------------------------ | ------------------------------------------------------------ |

**字源**：「え」【衣】	「エ」【江】

**发音**：「え」在发音时，双唇稍向左右咧开，舌面方平，口型和舌尖的位置处于「あ」和「い」之间。舌抬起，舌根用力，振动声带，声音紧张。【与英语中，『set, /set/』中的 /e/ 音相似】【扁唇音】

**书写**：「え」的左侧对齐；「エ」偏扁平。<span style>

**示例**：

-   上（う<span style="text-decoration: overline;">え</span>）【垂直正上方，物理位置；而非，抽象的在课本、报纸、黑板“上”的“上”】
-   家（い<span style="text-decoration: overline;">え</span>）【住宅，即：house；而非，家庭、家，即：home】

****

#### お (o) / オ (o)

| <img src="https://s2.loli.net/2024/10/18/y4Ha6WuswcIdErT.png" width="200px" height="200px"> | <img src="https://s2.loli.net/2024/10/18/t9jXIlg2sFwPvhz.png" width="200px" height="200px"> |
| ------------------------------------------------------------ | ------------------------------------------------------------ |

**字源**：「お」【於】「オ」【於】

**发音**：「お」的发音双唇稍微放圆，口型大小处于「あ」和「う」之间。舌面较平，振动声带。不像“欧”那样从口腔后部发出，声音更要圆润。【与英语中，『pop, /pɒp/』中的 /ɒ/ 音相似】【圆唇音】

**书写**：「お」左中下均对齐，第一笔的短横将右上角的点进行稍向上平分；「オ」正常书写【竖钩的钩有没有都可以】。

**示例**：青（あ<span style="text-decoration: overline;">お</span>）い【青色、绿色、蓝色的】

****

### 「さ」行假名

**字源**：「さ」【佐】 「サ」【散】

****

#### し (shi) / シ (shi)

****

#### す (su) / ス (su)

****

#### せ (se) / セ (se)

****

#### そ (so) / ソ (so)
